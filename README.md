## Installation

Add this line to your application's `Gemfile`

```ruby
gem 'blurred_image_tag'
```

and run

```ruby
bundle install
```

In your `application.css`, include the css file:

```css
/*
 *= require blurred_image_tag
 */
```

Then restart your webserver if it was previously running.
